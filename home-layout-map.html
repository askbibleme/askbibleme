主页版面结构分拆

读经首页对应仓库根目录 index.html，外层为 .page-shell；顶栏与底栏部分区域由 site-chrome.js 按接口数据注入。下列为静态骨架与主要 id，便于改样式或对接 main.js 时快速定位。

一、DOM 树概览

.page-shell
├── header.site-topbar
│   ├── .askbible-chrome-brand-slot          ← 品牌（接口注入）
│   ├── #openAdminBtn                         ← 程序化打开后台（通常隐藏）
│   └── .site-topbar-actions
│       ├── .askbible-chrome-nav-slot         ← 顶栏导航链接（接口注入）
│       └── #memberHub …                      ← 会员入口锚点
├── main.book-layout
│   └── .book-spread
│       ├── section.book-page.left-page
│       │   … 工具条 / 翻章 / 收藏带 …
│       │   ├── .page-title-block             ← 章号 + 书名大标题
│       │   ├── #chapterVideosSlot            ← 本章视频（大标题下、正文区上）
│       │   └── … 重复词、#leftBlocks …
│       └── section.book-page.right-page      ← 右页：副列 + 贡献区 + 底导航
└── footer.site-footer.askbible-chrome-footer-slot  ← 底栏（接口注入，默认可 hidden）

书页式排版由 styles.css 中 .book-layout / .book-spread 等类控制；窄屏下两栏会堆叠为单栏阅读流。

二、左页（left-page）主要区块

区域 关键选择器 / id 说明
阅读工具条 .page-toolbar、#bookChapterTrigger、#primaryVersionTrigger、#verseSearchOpenBtn、#displayModeSwitch 书卷 / 版本+ / 搜索 / 经文·全显·问题
隐藏表单（兼容 JS） #bookId、#chapter、#contentVersion 等 display:none 保留，供现有逻辑读写
章节标题与导航 .title-nav-row、#prevChapterBtnTop、#nextChapterBtnTop、#fontDecreaseBtn、#fontIncreaseBtn 顶栏翻章与字号
收藏与在线丝带 #sideFavoritesListBtn、#sideUserTag、#sideRibbonOnlineLine 收藏列表入口、用户标签、累计在线文案
本章视频（若有） #chapterVideosSlot 在大标题之前；有 chapterVideos 时渲染播放器，无则插槽为空不占位
本章标题 #pageChapterNumber、#pageBookTitle 大标题区
重复词行 #repeatedWordsLine 查经内容中的重复词提示
查经卡片主列 #leftBlocks（.blocks-stack） 半数字段卡片，与右列对半拆分

三、右页（right-page）主要区块

区域 关键选择器 / id 说明
副列查经卡片 #rightBlocks 与左列同属一段 segments，按序拆成两列
参与贡献 .chapter-feedback-wrap、#chapterQuestionInput、#submitChapterQuestionBtn 提交本章问题
社区已通过问题 #chapterApprovedTitle、#chapterApprovedQuestions 列表由接口数据渲染
底部章节导航 .chapter-nav.bottom-nav、#prevChapterBtnBottom、#nextChapterBtnBottom、#chapterNavTitleBottom 移动端友好的底栏翻章

四、全屏浮层（overlay）与模态框

均在 body 末尾、main.js 之前；多数为 .verse-search-overlay + 内层 role="dialog" 卡片，关闭按钮常带 data-close-panel="…"。

id 用途
#memberHubPanel 账户菜单（访客 / 已登录 / 千夫长入口）
#bookChapterPanel 书卷 + 章节网格（旧约 / 新约 / 章节）
#primaryVersionPanel 问题类型、主经文版本、对照版本
#favoritesPanel 收藏夹三标签（页面 / 经文 / 问题）
#verseSearchOverlay 经文关键词搜索
#authModal 登录 / 注册表单（.modal-mask）
#adminModal 读经内大面板：跳转独立管理页 + 规则编辑、测试生成等标签

五、脚本与配置入口

资源 作用
main.js 路由 hash、读经状态、渲染左右列、面板开关、管理员弹层等
site-chrome.js 顶栏品牌 / 导航、底栏内容注入（数据来自站点配置接口）
seo-apply.js 首页 SEO 元数据与 JSON-LD（可与后台 SEO 设置联动）
theme-apply.js 主题与字体缩放等全局外观

修改版面时建议同时检索 index.html、styles.css 与 main.js 中对上述 id 的引用，避免只改 HTML 导致脚本找不到节点。
