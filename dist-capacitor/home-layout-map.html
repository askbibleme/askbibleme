主页版面结构分拆

读经首页对应仓库根目录 index.html，外层为 .page-shell；顶栏与底栏部分区域由 site-chrome.js 按接口数据注入。下列为静态骨架与主要 id，便于改样式或对接 main.js 时快速定位。

是否「实时」？下方内置表格与 DOM 树是人工随仓库维护的，打开本页时不会自动扫描线上 DOM。只有「自定义模块」来自本机 localStorage。改版面后请同步改本页，或使用下面校验按钮拉取当前源的 /index.html，核对关键 id 是否已部署（需与读经同源，否则请用已部署站点打开本页再点）。 


      


      

        
一、DOM 树概览

        
.page-shell
├── header.site-topbar
│   ├── .askbible-chrome-brand-slot          ← 品牌（接口注入）
│   ├── #openAdminBtn                         ← 程序化打开后台（通常隐藏）
│   └── .site-topbar-actions
│       ├── .askbible-chrome-nav-slot         ← 顶栏导航链接（接口注入）
│       └── #memberHub …                      ← 会员入口锚点
├── main.book-layout
│   └── .book-spread
│       ├── section.book-page.left-page
│       │   … 工具条 / 翻章 / 收藏带 …
│       │   ├── .page-title-block             ← 章号 + 书名大标题
│       │   ├── #chapterIllustrationSlot      ← 章节插图（文前；有图时显示，无则 hidden）
│       │   ├── #chapterVideosSlot            ← 本章视频（插图下方；无视频则空）
│       │   └── … 重复词、#leftBlocks …
│       └── section.book-page.right-page
│           ├── #chapterVideosSlotRight         ← 宽屏时副列视频槽
│           ├── #rightBlocks                  ← 副列查经卡片
│           ├── #chapterEndPersonnelSection   ← 本章人员立绘横排（文末；卷首页 hidden）
│           ├── .chapter-feedback-wrap …      ← 参与贡献
│           ├── #chapterApprovedTitle / #chapterApprovedQuestions
│           ├── #chapterEndNextGuide            ← 「点击阅读下一章」引导（按需显示）
│           └── .chapter-nav.bottom-nav        ← 底栏翻章
└── footer.site-footer.askbible-chrome-footer-slot  ← 底栏（接口注入，默认可 hidden）

        

          书页式排版由 styles.css 中 .book-layout / .book-spread 等类控制；窄屏下两栏会堆叠为单栏阅读流。
        

      


      

        
〇、模块清单管理（本地）

        

          在下方「左页 / 右页」表格中可追加自定义模块行，仅保存在本机浏览器
          localStorage（换浏览器或清除站点数据会丢失），不会自动改动
          index.html 或线上版面。实际上线、改 DOM 仍需编辑仓库并部署。若将来需要「后台配置、全员同一套模块说明」，可把同类 JSON
          放到服务器，由管理接口读写，再在构建或运行时生成文档页（本页逻辑可复用为前端渲染器）。
        

        

          
          
          
        

      


      

        
二、左页（left-page）主要区块

        

          
        

        

          

            
              

                区域
                关键选择器 / id
                说明
              

            
            
              

                阅读工具条
                .page-toolbar、#bookChapterTrigger、#primaryVersionTrigger、#verseSearchOpenBtn、#displayModeSwitch
                书卷 / 版本+ / 搜索 / 经文·全显·问题
              

              

                隐藏表单（兼容 JS）
                #bookId、#chapter、#contentVersion 等
                display:none 保留，供现有逻辑读写
              

              

                章节标题与导航
                .title-nav-row、#prevChapterBtnTop、#nextChapterBtnTop、#fontDecreaseBtn、#fontIncreaseBtn
                顶栏翻章与字号
              

              

                收藏与在线丝带
                #sideFavoritesListBtn、#sideUserTag、#sideRibbonOnlineLine
                收藏列表入口、用户标签、累计在线文案
              

              

                本章标题
                #pageChapterNumber、#pageBookTitle
                大标题区（.page-title-block）
              

              

                章节插图（若有）
                
                  #chapterIllustrationSlot、.chapter-illustration-slot；内层由
                  main.js 注入 .chapter-illustration-figure /
                  .chapter-illustration-img
                
                
                  位于大标题下方、本章视频上方；当 JSON 含 chapterIllustration.imageUrl 时显示章节配图（默认生成为实底环境图；亦可为历史透明图），显示透明度受
                  /api/chapter-illustration/global-settings 全站配置；无图时
                  hidden 且不占布局高度
                
              

              

                本章视频（若有）
                #chapterVideosSlot
                在大标题与章节插图（若有）下方；有 chapterVideos 时渲染播放器，无则插槽为空
              

              

                重复词行
                #repeatedWordsLine
                查经内容中的重复词提示
              

              

                查经卡片主列
                #leftBlocks（.blocks-stack）
                半数字段卡片，与右列对半拆分
              

            
          

        

      


      

        
三、右页（right-page）主要区块

        

          
        

        

          

            
              

                区域
                关键选择器 / id
                说明
              

            
            
              

                副列查经卡片
                #rightBlocks
                与左列同属一段 segments，按序拆成两列
              

              

                本章人员（立绘+姓名）
                
                  #chapterEndPersonnelSection、.chapter-end-notes-table、#chapterEndPersonnelTableCaption、#chapterEndPersonnelRoster
                
                
                  位于右栏查经卡片之后、参与贡献之前。白底表格外壳，标题由 getLocalizedCopy().chapterEndPersonnelTableCaption 更新；单元格内
                  #chapterEndPersonnelRoster 为人物栅格（透明立绘 + 下方姓名）。
                  main.js 中 syncChapterCharacterRoster() 根据
                  /api/study-content（及兜底 /api/study-character-figures）的
                  chapterCharacterFigures 渲染；名单由服务端 keyPeople 与人物库肖像匹配。无人物时表仍在、单元格为空；仅卷首页
                  hidden（且 .book-layout--landing 下 CSS 隐藏）。
                
              

              

                参与贡献
                .chapter-feedback-wrap、#chapterQuestionInput、#submitChapterQuestionBtn
                提交本章问题
              

              

                社区已通过问题
                #chapterApprovedTitle、#chapterApprovedQuestions
                列表由接口数据渲染
              

              

                下一章阅读引导
                
                  #chapterEndNextGuide、#chapterEndNextHint、#chapterEndNextBtn、#chapterEndNextBtnLabel
                
                居中提示「点击阅读下一章」与跳转下一章（由脚本按场景显示/隐藏）
              

              

                底部章节导航
                .chapter-nav.bottom-nav、#prevChapterBtnBottom、#nextChapterBtnBottom、#chapterNavTitleBottom
                移动端友好的底栏翻章
              

            
          

        

      


      

        
四、全屏浮层（overlay）与模态框

        

          均在 body 末尾、main.js 之前；多数为 .verse-search-overlay + 内层
          role="dialog" 卡片，关闭按钮常带 data-close-panel="…"。
        

        

          

            
              

                id
                用途
              

            
            
              

                #memberHubPanel
                账户菜单（访客 / 已登录 / 千夫长入口）
              

              

                #bookChapterPanel
                书卷面板：>480px 三列（旧约｜新约｜章节）；≤480px 两列合并+书卷简称
              

              

                #primaryVersionPanel
                主经文版本、对照版本、问题类型
              

              

                #favoritesPanel
                收藏夹三标签（页面 / 经文 / 问题）
              

              

                #verseSearchOverlay
                经文关键词搜索
              

              

                #authModal
                登录 / 注册表单（.modal-mask）
              

              

                #adminModal
                读经内大面板：跳转独立管理页 + 规则编辑、测试生成等标签
              

            
          

        

      


      

        
五、脚本与配置入口

        

          

            
              

                资源
                作用
