顶栏与底栏

配置写入 admin_data/site_chrome.json，全站通过 site-chrome.js 统一渲染；管理后台左侧 LOGO也读同一字段（浅色侧栏会自动把 askbible-wordmark.svg 换成深色字版 askbible-wordmark-ink.svg）。主页顶栏默认使用站内横版透明字标 /assets/brand/askbible-wordmark.svg（无需上传、勿复制外链）；若要侧栏与顶栏都用深色字标，可把下方地址直接改为 /assets/brand/askbible-wordmark-ink.svg。亦可改为其他本站路径或 https 地址。

本页修订 2026-04-11：「显示」与复选框同属可点标签；请点复选框或「显示」文字切换（只点格子外空白不会改状态）。若界面异常请先 Unregister Service Worker 并硬刷新。本地请用 node server.js 打开当前仓库。

顶栏设置

左侧为品牌区（LOGO 与主标题），其旁可配置网页一句话介绍（顶栏标语，如「读经平台 · 在线查经 · 小组聚会」）；右侧导航在下方「顶栏导航链接」中逐项填写。保存后全站顶栏一致。

侧栏浅色背景推荐 /assets/brand/askbible-wordmark-ink.svg；若图片裂了会自动改用应用图标 PNG 再退回「Ask Bible .me」文字。

网页介绍（顶栏标语）

访客在顶栏看到的站点说明，与品牌名并列或位于其下；用于一句话概括本站（纯文本，勿填 HTML）。

最多 120 字。勾选「显示」后，读经主页等所有带顶栏的页面都会载入该文案。

顶栏导航链接

「设置」：新增一行，链接填 /#openReaderFontSettings，图标选「设置」，勾选「仅图标」；最左「显示」可隐藏。若希望出现在页面最底部，请用下方「底栏图标链接」同样配置并打开「显示全站底栏」。「下一章 / 上一章」与书页内箭头一致：链接填 /#goNextChapter、/#goPrevChapter（须在读经页且存在下一章/上一章时才会跳转）。

每一项对应顶栏右侧一个深色圆角标签。左侧可拖动 ⋮⋮ 或按 ↑↓ 调整顺序。「图标」用下拉菜单选择（左侧小框为当前预览）；勾选 仅图标 后顶栏只显示图标（须先选图标，读屏仍朗读显示文字）。链接须以 / 或 https:// 开头；最多 16 条。 会员书签（「昵称的圣经」下拉菜单）本身不是普通超链接；若要在顶栏加一项同样效果，请填链接地址 /#openMemberHub（进入读经首页并自动打开会员面板）。宣传页等同理可用 /why.html#openMemberHub。账户与登录相关请使用会员面板。

底栏设置

可选全站底部说明文字（纯文本、按行分段，不支持 HTML）与图标链接行。文字分为左 / 中 / 右三栏；图标链接与顶栏同款圆角标签，显示在文字下方、整行居中。仅填中间一栏文字时接近原先「整段居中」效果。

底栏图标链接

与顶栏导航相同：选图标、可勾选「仅图标」；链接支持 /、https:// 与 /#openReaderFontSettings 等同页动作。不要往文字栏里粘贴 HTML，图标按钮请在这里配置。

最多 16 条，与顶栏合计独立计数。取消「显示」可隐藏单项。
